# Used Car Price Prediction & Transmission Classification

Machine learning project for predicting used car prices (Regression) and transmission types (Classification).

## Dataset
- cars.csv

## Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Regression modeling
- Classification modeling
- Cross-validation
- Performance evaluation (F1-score, accuracy, etc.)

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Files
- project1_cars.py → Main modeling script
- project1_plots.py → Visualization script
- 261_project_report.pdf → Project report
