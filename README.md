# Used Car Price Prediction & Transmission Classification

Machine learning project for predicting used car prices (Regression) and transmission types (Classification).

## Dataset
- cars.csv

## Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Regression modeling
- Classification modeling
- Cross-validation
- Performance evaluation (F1-score, accuracy, etc.)

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## How to Run
- Install dependencies
   pip install pandas numpy scikit-learn matplotlib

 - Run model training
   python model_training.py

 - Run visualization
   python visualization.py

## Files
- model_training.py → Regression & classification training pipeline
- visualization.py → Data visualization & EDA script
- project_report.pdf → Project report
